## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

The app will open at `http://localhost:3000`

## Usage

1. Make sure the backend server is running at `http://localhost:5000`
2. Enter a student roll number in the search box
3. View comprehensive attendance analytics and insights

## Technologies

- React 18
- Tailwind CSS
- Recharts (for data visualization)
- Axios (for API calls)
